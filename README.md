## 📖 Overview

**Smart Utility Suite** is a sleek, production-ready web application that bundles seven essential productivity tools into a single, unified dashboard. Built with React 19 and Vite, it delivers a premium SaaS-grade experience — complete with smooth micro-animations, responsive layouts, and an intuitive sidebar navigation — so you never have to juggle multiple apps for everyday digital tasks again.

---

## ✨ Features

| Tool | Description |
|------|-------------|
| 📊 **EMI Calculator** | Calculate monthly installments with interactive charts powered by Chart.js |
| 🧾 **GST Calculator** | Instantly compute GST amounts for any tax slab |
| 💱 **Currency Converter** | Real-time currency conversion with live exchange rates |
| 🔐 **Password Generator** | Generate secure, customizable passwords with strength indicators |
| 📱 **QR Code Generator** | Create downloadable QR codes from any text or URL |
| 🧹 **JSON Formatter** | Beautify, minify, and validate JSON with syntax highlighting |
| 📝 **Word Counter** | Analyze text with word count, character count, and reading time |

### Highlights

- 🎨 **Premium UI** — Stripe / Linear-inspired design with glassmorphism accents
- 📱 **Fully Responsive** — Seamless experience across desktop, tablet, and mobile
- ⚡ **Blazing Fast** — Vite-powered build with instant HMR
- 🧭 **Sidebar Navigation** — Switch tools without page reloads
- 📈 **Data Visualization** — Interactive charts for the EMI calculator
- 🔄 **Copy & Reset** — One-click copy and reset on every tool

## 🚀 Getting Started

### Prerequisites

- **Node.js** ≥ 18
- **npm** ≥ 9

### Installation

```bash
# Clone the repository
git clone https://github.com/Vanshika-v13/Smart-Suite.git

# Navigate into the project
cd Smart-Suite

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be running at **`http://localhost:5173`**.

### Build for Production

```bash
npm run build
npm run preview
```

---

## 💡 Usage

1. Open the app in your browser.
2. Use the **sidebar** to navigate between the seven utility tools.
3. Enter your inputs, hit calculate / generate, and get instant results.
4. Use the **Copy** button to copy outputs and **Reset** to clear fields.
